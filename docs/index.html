About this app

SipSmartAI helps you log alcohol intake, track BAC-level trends, and earn safe-drinking “smart points” with clear guidance based on your tolerance. It is built as a harm-reduction tool for people who want more awareness and control over their drinking—not as a moral judgment.

  • Set a personal tolerance and see your BAC progress in real time.
  • Log drinks with defaults for beer, wine, and shots or add custom pours.
  • View weekly “Smart Points” to see how often you stay within safe limits.
  • Use reminders and notifications to avoid going past your set threshold.

Key Features

  • Golden Path flow to set up profile, tolerance, and first session.
  • Smart BAC progress bar with your limit clearly displayed.
  • Weekly, monthly, and yearly stats pages powered by Firebase.
  • Survey + NPS to collect user feedback for future iterations.

Tech Stack

  • FlutterFlow front-end (Flutter / Dart).
  • Firebase Auth & Firestore for data and accounts.
  • Firebase Analytics & Remote Config for metrics and A/B tests.
  • GitHub + GitHub Pages for this marketing site.
